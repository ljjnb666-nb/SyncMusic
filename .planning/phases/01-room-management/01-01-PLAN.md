<!-- Key types and contracts extracted from research -->

Room State Schema (from ARCHITECTURE.md):
```javascript
{
  id: string,              // nanoid(10) URL-safe room ID
  name: string,            // User-provided room name
  createdAt: timestamp,
  hostId: string,          // Session ID of room creator
  isPlaying: boolean,
  currentTrack: null,      // Phase 2
  position: 0,
  positionUpdatedAt: timestamp,
  playlist: [],
  participants: [{
    id: string,           // nanoid(8) session ID
    username: string,
    joinedAt: timestamp,
    isHost: boolean
  }]
}
```

REST Endpoints:
- POST /api/rooms - Create room, returns { roomId, inviteUrl }
- GET /api/rooms/:id - Get room metadata

Socket.io Events:
- Server emits: room:state, room:join, room:leave, room:host
- Client emits: room:join
</interfaces>

  <action>
Create backend/src/socket/handlers.js with these exported functions:

1. `handleJoin(io, socket, { roomId, username })`:
   - Validate room exists via RoomManager
   - If not found, emit 'error' event with message 'Room not found'
   - Generate userId = nanoid(8) if not in socket.data
   - Create participant: { id: userId, username, joinedAt: Date.now(), isHost: false }
   - Add to room.participants array
   - Store socket.data.roomId = roomId, socket.data.userId = userId
   - socket.join(roomId)
   - Emit 'room:state' to joining user with FULL room object (first, before others)
   - socket.to(roomId).emit('room:join', { userId, username }) to others

2. `handleDisconnect(io, socket, reason)`:
   - Get roomId and userId from socket.data
   - If no roomId, return (wasn't in a room)
   - Get room from RoomManager
   - Find and remove participant from room.participants array
   - socket.to(roomId).emit('room:leave', { userId })
   - If user was host AND room.participants.length > 0:
     - Sort remaining by joinedAt ascending (oldest first)
     - newHost = room.participants[0]
     - newHost.isHost = true
     - room.hostId = newHost.id
     - io.to(roomId).emit('room:host', { hostId: newHost.id })
   - If room.participants.length === 0:
     - RoomManager.delete(roomId)
     - db.deleteRoom(roomId) - cleanup persistence
   - socket.leave(roomId)

3. Export { handleJoin, handleDisconnect }
</action>

    <automated>node -e "const { handleJoin, handleDisconnect } = require('./backend/src/socket/handlers.js'); console.log('Handlers exported:', typeof handleJoin, typeof handleDisconnect)"</automated>
  </verify>
  <acceptance_criteria>
    - handleJoin: validates room, adds participant, emits room:state to joiner, room:join to others
    - handleDisconnect: removes participant, transfers host if needed (oldest participant becomes new host), cleans up empty rooms
    - Host transfer: sorts by joinedAt ascending, assigns isHost: true to first
    - Empty room cleanup: removes from both RoomManager and SQLite
  </acceptance_criteria>

# SyncMusic - 一起听音乐

基于 Vue + Socket.IO 的实时音乐同步应用。

## 功能

- 音乐解析下载 (QQ音乐/网易云/酷狗)
- 一起听房间 (多人实时同步播放)
- 本地音乐文件夹扫描
- 音乐文件直接导入
- 多平台音乐搜索

## 快速开始

### 后端
```bash
cd backend
npm install
node src/index.js
```

### 前端
```bash
cd frontend
npm install
npm run dev
```

## 技术栈

- Frontend: Vue 3, Vite, Socket.IO Client
- Backend: Node.js, Express, Socket.IO
